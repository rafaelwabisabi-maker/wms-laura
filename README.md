# WMS Laura

A lightweight warehouse management system for small-scale inventory tracking.

## About

WMS Laura is a mini-WMS built for small businesses that need inventory control without the complexity of enterprise solutions. It handles product cataloging, stock movements, picking workflows, and basic reporting. Designed to be self-hosted and easy to operate for non-technical warehouse staff.

## Tech Stack

- Next.js / React
- TypeScript
- Prisma ORM / SQLite
- Tailwind CSS
- Docker
